Listas

Existem 3 tipos de Listas dentro do HTML

Lista Não Ordenadas

é uilizado as tags UL abrindo a lista e, para cada item era a tag LI

  • item
  • item
  • item

Lista Ordenadas

é uilizado as tags OL abrindo a lista e, para cada item era a tag LI

  1. item
  2. item
  3. item

Lista de Definição

é uilizado as tags DL abrindo a lista, DT para informar o título e DD para item, como em um sumario

titulo
subtitulo 1
subtitulo 2

----- O que é OOP ----
OOP(Oriented Object Programming) é um programa que tem varios objetos com regras especificas seguidas 
que visa atender demandas grandes ou criar sistemas completos com configuraçoes diferentes para um
mesmo objeto

----- BASE pt1 ----
Termos:
*classe = 
        Classes são a mesma coisa que objetos, são um conjunto de funcionalidades que podem ser reutilizadas.
*objetos =
        Cada Classe é um Objeto, pois ela visa ser util em algum tipo de aspecto, se nao me engano, sendo baseado 
        em um objeto realistico, como um controle remoto
*atributos =
        caracteristicas, configuraçoes que todas as instancias de seu codigo precisa ter. Coisas como
        ex: classe de clientes. Todo cliente precisaria ter atributos como nome, idade, cpf
        mas indo de acordo com a utilização da sua classe
*metodos =
        funcionalidades (funções mesmo) do objeto separadas em funcoes para interagir com a classe.
        ex: aumentar volume, trocar de canal, etc
*instancias =
        são objetos baseados na classe, normalmente para criar objetos semelhantes com atributos diferentes
        logo as açoes ao decorrer do objeto seriam diferentes das demais, devido as suas configuraçoes 
*__init__ =
        é o nome da função inicial da classe. É onde os atributos / configuraçoes da classe ficam
*self =
        utilizado dentro da função __init__ pois visa apresentar que os valores daquelas configurações 
        poderiam ser utilizados pelas instancias, evitando que todas as instancias tivessem as mesmas config

------ CODIGO EXEMPLO ----
# ATRIBUTOS
class Pessoas:
    def __init__(self, user, nome, senha, plano):
        self.user = user
        self.nome = nome
        self.senha = senha
        self.lista_planos = ["basico", "estudantil", "premium"]
        if plano in self.lista_planos:
            self.plano = plano
            print(f"Plano {plano}")
        else:
            print("Plano inexistente")

    def trocar_plano(self, novo_plano):
        if novo_plano in self.lista_planos: #novo_plano nao tem self porque ta fora da função __init__
            self.plano = novo_plano
            print(f"Plano atualizado para {self.plano}")
        else:
            print("Plano inexistente")



#METODOS :

pessoa1 = Pessoas("chromantico", "jamily", "1234", "premium")
print(pessoa1.user)
print(pessoa1.plano)
pessoa1.trocar_plano("estudantil")
print(pessoa1.plano)
print("--"*4)
